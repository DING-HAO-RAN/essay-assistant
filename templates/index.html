使用说明

如何获取API Key

支持的服务提供商

  • OpenAI：访问 platform.openai.com 注册并获取API Key
  • Claude：访问 console.anthropic.com 注册并获取API Key
  • DeepSeek：访问 platform.deepseek.com 注册并获取API Key
  • Moonshot (Kimi)：访问 platform.moonshot.cn 注册并获取API Key
  • 智谱AI (GLM)：访问 open.bigmodel.cn 注册并获取API Key
  • 通义千问：访问 dashscope.aliyun.com 注册并获取API Key

配置步骤

  1. 选择您要使用的服务提供商
  2. 输入对应的API Key
  3. 选择要使用的模型（不同模型效果不同）
  4. 点击"测试连接"确认配置正确
  5. 点击"保存配置"完成设置

注意事项

  • API Key是您的私密凭证，请勿泄露给他人
  • 不同服务商的API调用可能产生费用，请注意用量
  • 推荐使用DeepSeek或智谱AI，性价比高且中文效果好
